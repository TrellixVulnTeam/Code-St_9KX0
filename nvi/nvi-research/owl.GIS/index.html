Мобильные приложения, серверные решения (веб сервисы) и фронт-офисные клиентские приложения.

Прикладные микросервисы

Сервисы, предназначенные для выполнения функций, предусмотренными ТЗ

Zuul

API шлюз, предназначен для переадресации запросов от приложений системы на соответствующие сервисы, с проверкой прав пользователей

Keycloak

Сервис авторизации, позволяющий выполнять авторизацию с использованием LDAP (Active Directory), соцсетей, ЕСИА;

Cloud config

Единый сервис управления настройками микросервисов, а также интеграционных модулей

Geoserver

Геоинформационный сервер, реализующий поддержку стандартов Open Gis Consortium, таких как WMS, WFS, WMTS, а также векторных тайлов, использующимися в прогрессивном рендеринге карт и схем на клиентских устройствах.

Postgesql

Предназначены для получения данных из систем заказчика

Redis

Сервисы, предназначенные для выполнения функций, предусмотренными ТЗ

HASURA

API шлюз, предназначен для переадресации запросов от приложений системы на соответствующие сервисы, с проверкой прав пользователей

NGINX

Сервис авторизации, позволяющий выполнять авторизацию с использованием LDAP (Active Directory), соцсетей, ЕСИА;

Elasticsearch
