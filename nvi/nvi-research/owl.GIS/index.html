Возможности продукта

Отображает картографические материалы в режиме 3D, включая детальные 3D модели различных объектов

Работает с неограниченным по территориальному охвату и наполнением пространственными данными

Интегрируется с системами мониторинга строительства и персонала, бурения, видеомониторинга, АСУТП и др.

Использует в качестве клиентского ПО стандартный web-браузер

Возможности аналитики

Выполнение оверлейных запросов

Автоматическая классификация, оцифровка данных с использованием нейронных сетей собственной разработки

Конструирование алгоритмов обработки растровых и векторных операций

Динамическое построение тепловых карт

Формирование аналитической отчетности по совокупности данных

Сервисы, предназначенные для выполнения функций, предусмотренными ТЗ

Zuul

API шлюз, предназначен для переадресации запросов от приложений системы на соответствующие сервисы, с проверкой прав пользователей

Keycloak

Сервис авторизации, позволяющий выполнять авторизацию с использованием LDAP (Active Directory), соцсетей, ЕСИА;

Cloud config

Единый сервис управления настройками микросервисов, а также интеграционных модулей

Geoserver

Геоинформационный сервер, реализующий поддержку стандартов Open Gis Consortium, таких как WMS, WFS, WMTS, а также векторных тайлов, использующимися в прогрессивном рендеринге карт и схем на клиентских устройствах.
