/owl.Scan

Сетевое API для оптического распознавания документов

Обсудить детали Свидетельство правообладателя

О продукте

Универсальное сетевое API для развертывания сервиса оптического распознавания документов на стороне клиента. /owl.Scan предназначен для решения широкого спектра практических задач автоматизации процессов дистанционного обслуживания клиентов.

Быстро

Сервис позволяет c высокой скоростью распознавать различные классы документов, а также проводить проверку подлинности удостоверяющих документов и сверку лиц

Безопасно

Сервис устанавливается в контуре заказчика без передачи персональных данных сторонним сервисам и третьим лицам

API

Распознающий сервис ведет обмен данными в открытом формате JSON используя в качестве транспорта HTTP протокол, позволяя интегрироваться с ним из любого технологического стека.

API имеет описание в формате OpenAPI и поставляется вместе с инструментами документации и демонстрационным приложением.

Служба распознавания может быть дополнена сервисом хранения распознанных документов, который может использовать как внутреннее, так и внешнее хранилище данных.

Историю можно использовать для поиска, просмотра и анализа распознанных документов.

Сервис осуществляет управление распознающими сессиями. В рамках каждой сессии можно обработать документ в режиме single-shot, имея только одно изображение или использовать режим агрегированного результата, при котором качество распознавания можно улучшать, используя различные изображения документа.

В рамках одной распознающей сессии можно распознавать информацию с различных сторон документа. Например, разных страниц паспорта или передней и задней стороны прав.

Проверки подлинности
и сверка лиц

Для особо чувствительных документов, таких как паспорта, права или кредитные карты, система осуществляет проверку признаков подлинности.

Например, для паспорта, в оптическом диапазоне, осуществляется проверка по 32 различным признакам.Результаты этих проверок используются для организации защиты от мошенничества (anti-fraud).

В дополнении к признакам подлинности, система может проводить сверку лиц в документе и фотографии предъявителя.

Результат проверки — процентное значение уровня схожести которое может быть использовано как основание для дополнительной, детальной проверки документа.

Распознающий модуль не выделяет никаких биометрических признаков, сверка лиц максимально приближена к тому, как это делает человек. Отсутствие устойчивых дескрипторов лиц не позволяет ни кластеризовать, ни распознавать лица по базе.
