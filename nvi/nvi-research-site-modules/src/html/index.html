/owl.Guard

Анализирует
видео онлайн

Выявляет
нарушения

Комплекс программных модулей для автоматического распознавания нарушений на производстве.

Программный комплекс автоматически анализирует видеопоток с камер видеонаблюдения, установленных на производственных площадках заказчика, и выявляет нарушения правил промышленной и производственной безопасности, правил эксплуатации технологического оборудования.

QR, банковские карты

Проверяет
подлинность

Универсальное сетевое API для развертывания сервиса оптического распознавания документов на стороне клиента. /owl.Scan предназначен для решения широкого спектра практических задач автоматизации процессов дистанционного обслуживания клиентов.

Программные модули распознавания документов для мобильных приложений, серверных и фронт-офисных решений.

/owl.GIS

Сбор, анализ, хранение,
3D визуализация данных

В режиме
реального времени

Геоинформационная система для сбора, обработки, анализа и хранения данных с производственных объектов в режиме реального времени.

Обеспечивает сбор данных от корпоративных и технологических информационных систем в режиме реального времени и позволяет автоматизировать решение аналитических и управленческих задач.

/owl.Tag + Anchor

Контроль
местонахождения
персонала

Контроль
здоровья
сотрудника

Программно-аппаратный комплекс позиционирования персонала на производственных объектах в условиях покрытия сети LoRaWAN и UWB.

Контролирует нахождения персонала на объектах заказчика (на открытых площадках, в помещениях). Контроль активности персонала и состояния его здоровья. Интеграция с носимыми устройствами через Bluetooth для снятия показателей здоровья
